# Anki-карточки для вступительного экзамена по философии в аспирантуру

Файл `Философия (Павлова).apkg` содержит карточки для приложения [Anki для ПК](http://ankisrs.net/) (и [AnkiDroid](https://play.google.com/store/apps/details?id=com.ichi2.anki&hl=ru)), составленные по некоторым [видеолекциям по философии](http://www.youtube.com/playlist?list=PLKU78QmFP1FmeOPfi9bQ6BEnsAC2uBomN) [Павловой Елены Леонидовны](http://www.philosophical.ru/).
